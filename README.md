# FarmTech Solutions

Repositorio organizado por fases.

## Estrutura

- `fase1/`: entrega da fase 1, separada por tipo de arquivo.
- `fase2/`: entrega da fase 2, com codigo, simulacao e documentacao.
- `README.md`: indice geral do repositorio.

## Como ler o repositorio

Cada fase e autonoma dentro da propria pasta. Dentro dela, os arquivos ficam separados por finalidade:

- `python/` para scripts em Python
- `r/` para analises em R
- `esp32/` para codigo e simulacao do ESP32
- `dados/` para CSVs e outras bases de apoio
- `docs/` para documentos, diagramas e evidencias
